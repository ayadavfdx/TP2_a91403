# Stock Portfolio Manager

## Requirements
Python 3.10+

Libraries used:
- **colorama**
- **plotly**
- **yfinance**

## Installation
```bash
pip install colorama plotly yfinance
```

## Description
This project is a simple stock portfolio manager implemented in Python as part of an Object-Oriented Programming exercise.

## Features
- Manage a stock position
- Buy shares
- Sell shares
- Calculate weighted average purchase price
- Calculate potential profit
- Receive dividends
- Colored console output using **colorama**
- Interactive stock chart using **plotly**

## OOP Concepts Applied
- **Encapsulation** 
- **Data validation** 

## How to Run

Run the main file from the terminal:

```bash
python tp2_main.py
